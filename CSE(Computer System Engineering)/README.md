# 计算机系统工程(CSE)

计算机系统工程这门课中我们主要完成了4次Lab以及四次小作业（hands on），其中：
+ Lab1中主要完成了一个文件系统（ChFS）的编写
+ Lab2中完成了一个MapReduce的工具，通过RPC调用实现了分布式系统上的单词数量统计
+ Lab3中完成了一个基于Raft协议的Key-Value存储，其中的PART4未完全实现
+ Lab4中完成了一个分布式支持事务的Key-Value存储
+ Hands on 1&2中通过实际操作以及问题思考了解了WAL
+ Hands on 3中通过实际操作了解了DNS
+ Hands on 4中通过分析TCP传输记录了解了TCP传输
